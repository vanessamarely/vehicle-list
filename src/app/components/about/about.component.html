This project was generated with [Angular CLI 1.6.2 version] and Angular 5 version.

At home page, a user can search by Brand. I added a filter option to user can filter using drop-down box

To check a Vehicle page user must click on Model name

To compare Vehicles by Model, user can click on button next to Model name and when user selected at least two options to compare button will be displayed and if user clicked on it will be redirected to compare section.
